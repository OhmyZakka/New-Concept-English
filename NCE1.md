# Lesson 5~6

#### 语法知识点

- 冠词

概念：冠词是用于名词前的虚词，用来表示名词是特指还是泛指。

包括：a / an / the

意思：a / an 一个、一台、一间、一 ...

   	the 这，那，这些，那些

​	   the 后面是单数可译为这，那

​	   the 后面是复数可译为这些，那些

分类：**定冠词** the  表示确定的、知道的

​	    **不定冠词** a / an 表示不确定的、不知道的

​	    **零冠词** 国家、国籍、人名、地名等专有名词不需要加冠词。

a / an 的区别：an + 元音发音开头的单词，其余用a 例如：an egg / an hour

​
