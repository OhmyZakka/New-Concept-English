

# Lesson 1~2

#### 语法知识点

- 一般疑问句的构成形式

一般疑问句

一般疑问句功能：询问

一般疑问句构成：be动词置于句前

一般疑问句将be动词提前，即将一般陈述句的be动词提前到句首

be --- is,am,are



- 简单句的常用结构

**主+谓+宾**

为什么英文会有主格和宾格之分？当一个词做主语的时候用主格，做宾语的时候就用宾格。

- 人称代词的主格和宾格

| 宾格 | 主格 |
| ---- | ---- |
| me   | I    |
| you  | you  |
| her  | she  |
| him  | he   |
| it   | it   |

#### 重点词汇

- it 它

可代替物体、时间、天气

It is a umbrella.

It is 8:00.

It is sunny today.
